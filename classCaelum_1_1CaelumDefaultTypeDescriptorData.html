Detailed Description

Standard type descriptors for caelum components.

This class hold pointers to several type descriptors for classes inside Caelum. All the pointers are initialize in the contructor and properly destroyed in the destructor.

The CaelumPlugin singleton contains a const instance of this class. You should fetch that instead of creating a new object; using CaelumPlugin::getTypeDescriptorData().
